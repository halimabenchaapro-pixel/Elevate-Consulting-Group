# Elevate Consulting Group

A modern, fully responsive corporate consulting website built with pure HTML & CSS.

## Pages

- **Home** — Hero, services overview, testimonials, case study previews
- **About Us** — Company story, values, and team
- **Services** — Detailed service offerings and pricing
- **Case Studies** — Client results and success stories
- **Contact** — Booking form and company info

## Tech Stack

- HTML5
- CSS3 (custom, no frameworks)
- Google Fonts (Inter + Playfair Display)

## Features

- Fully responsive (mobile, tablet, desktop)
- Sticky navbar with scroll effect
- Professional blue/gray/white color scheme
- Smooth animations and hover effects
